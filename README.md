# PC微信读书评论插件
支持浏览器场景访问微信读书页面时，加载和显示读书评论，提升阅读体验。

## 一期: 2023-12~2024-01
请求微信评论接口数据，侧边栏渲染评论数据，支持评论引用原文的展开和收起
